# Drum Sound

An interactive web-based drum machine that plays sounds on key press or click.

## Tech Stack
- HTML
- CSS

## Getting Started
```bash
git clone https://github.com/mrsandeep27/Drum-sound.git
cd Drum-sound
```

Open `ind.html` in your browser.

## Project Structure
```
ind.html    # Main application
style.css   # Styling
sound/      # Audio files
```

## Features
- Interactive drum pad UI
- Audio playback on key press / click
- No frameworks or build tools needed
